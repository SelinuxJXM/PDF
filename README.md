# PDF墨韵 — 本地PDF工具箱

> 🛡️ **100% 浏览器本地处理，文件永不离开你的设备**

[在线使用 →](https://unej4622-cpu.github.io/pdf-studio/)

---

## 功能

| 工具 | 说明 |
|------|------|
| 🧩 合并 PDF | 上传多个 PDF，按顺序合并为一个文件 |
| ✂️ 拆分 PDF | 按页面范围拆分，或将每一页单独导出为 ZIP |
| 🔄 旋转页面 | 选中页面后，90° / 180° / 270° 任意旋转 |
| 🗑️ 删除页面 | 点击选中页面，一键删除多余页面 |
| ↕️ 重排顺序 | 拖拽页面卡片，灵活调整页面顺序 |

## 特点

- ✅ **完全免费** — 无需注册，无使用次数限制
- ✅ **隐私安全** — 所有处理在浏览器本地完成，文件不经过任何服务器
- ✅ **双主题** — 支持浅色 / 深色模式切换
- ✅ **无需安装** — 直接在浏览器打开即可使用
- ✅ **离线可用** — 加载后可在无网络环境下使用

## 技术

- PDF 渲染：pdf.js (Mozilla)
- PDF 操作：pdf-lib
- 打包下载：JSZip
- 纯前端实现，无需后端服务器
